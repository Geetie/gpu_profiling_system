#### 3.1 立即删除 `_generate_skeleton` 方法

**理由**：

1. 违反核心原则
2. 无法完成真实测量（只有空循环）
3. 破坏 LLM-as-a-Judge 评估机制

**替代方案**：

```python
def _generate_kernel(self, target: str, category: str, method: str) -> str:
    """Generate CUDA kernel source code via LLM.

    If LLM is unavailable, FAIL GRACEFULLY — do not fallback to hardcoded code.
    """
    if self._model_caller is not None:
        messages = self.context_manager.to_messages()
        try:
            return self._model_caller(messages)
        except Exception as e:
            self._persister.log_entry(
                action="llm_call_failed",
                details={"error": str(e), "target": target},
            )
            # ✅ 优雅失败：抛出异常，而非硬编码代码
            raise RuntimeError(f"LLM unavailable for code generation: {e}")
    
    # ✅ 明确错误：LLM 未配置
    raise RuntimeError("No LLM configured — CodeGen requires LLM to generate CUDA code")
```
